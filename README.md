## 📖 Sobre o Projeto

O **Jujutsu Kaisen Personagens** é um projeto desenvolvido com **HTML5 puro**, com o objetivo de apresentar personagens do universo do anime de forma organizada e interativa.

Além disso, o projeto funciona como um **laboratório prático de HTML**, explorando:

- Estrutura semântica  
- Navegação por âncoras  
- Organização de conteúdo em seções
